TROUBLESHOOTING:
================

Camera Not Working?
- Check camera permissions
- Close other apps using camera
- Click "Test Camera" button

Port Already in Use?
- Stop other Flask apps
- Or change port in app.py

Dependencies Error?
- Run: pip install -r requirements.txt

Page Loading Slow?
- Clear browser cache
- Restart application
- Check internet connection
